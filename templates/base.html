For detection of Depression in patients, data can be uploaded manually or autogenerated.

Data Format Table

Type of EEG data to be uploaded Number of files to be uploaded Extension of files to be uploaded

*Please refer to Data Format Table for details regarding data*

To detect depression using recorded EEG data-

  • Click on the “Upload files” and upload data
  • Click on the “Predict” button that appears after uploading data
  • Result will be displayed



To detect depression using auto generated data-

  • Enter the time(in seconds) according to limit
  • Click on "Submit" button
  • Result will be displayed
